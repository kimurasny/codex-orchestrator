# 画面仕様書生成プロンプト

あなたは業務システムの設計者です。
以下のソースファイルを解析し、画面仕様書を Markdown 形式で出力してください。

## 対象

- 対象ファイル(相対パス): {{TARGET}}
- 絶対パス: {{ABS_PATH}}
- ファイル名: {{FILE_NAME}}
- プロジェクトルート: {{PROJECT_ROOT}}
- 出力先: {{OUTPUT}}

## 出力要件

1. 画面概要（画面名・目的）
2. 画面項目一覧（項目名・種別・必須有無・初期値・バリデーション）
3. イベント一覧（操作・トリガ・処理内容）
4. 画面遷移（遷移元・遷移先・条件）
5. 呼び出す業務ロジック / API
6. 補足・留意事項

生成した仕様書は Markdown 形式で、出力先 {{OUTPUT}} にファイルとして保存してください（余計な前置きは含めないでください）。
